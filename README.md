# Supervity AI Command Center 🚀

Supervity is a scalable AI Agent orchestration platform designed to function as a centralized AI Command Center for managing intelligent workflows, API integrations, and real-time agent communication.

The platform enables seamless interaction between AI agents, backend services, automation systems, and modern frontend interfaces through a production-ready full-stack architecture.

---

## 📌 Overview

Supervity acts as a bridge between AI-powered systems and real-world applications by providing:

- Centralized AI workflow management
- Agent communication orchestration
- API integration handling
- Real-time backend processing
- Scalable frontend monitoring dashboard
- Dockerized deployment infrastructure

The backend communicates with Supervity AI APIs and processes intelligent agent operations while the frontend provides an interactive command center interface for monitoring and controlling workflows.

## ⚙️ Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Supervity.git
cd Supervity
```

---

## 🔐 Environment Configuration

### Backend

Create a `.env` file using the example template:

```bash
cp .env.example .env
```

### Frontend

Create a `.env.local` file using the example template:

```bash
cp .env.local.example .env.local
```

---

## 🐳 Running the Application

### Backend

```bash
cd Backend
docker-compose up --build
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```
